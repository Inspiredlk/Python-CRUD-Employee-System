# 🚀 Employee Management System (Python CRUD)

A simple and responsive **Employee Management System** built using **Python** and **Flask**. This application allows users to perform complete **CRUD (Create, Read, Update, Delete)** operations for employee records through a clean web interface.

---

## ✨ Features

- ➕ Add New Employees
- 📋 View Employee Records
- ✏️ Update Employee Details
- 🗑️ Delete Employees
- 💾 SQLite Database Support
- 📊 Export Employee Records to Excel
- 🎨 Clean & Responsive User Interface

---

## 🛠️ Technologies Used

- Python
- Flask
- SQLite
- HTML5
- CSS3
- JavaScript

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/Inspiredlk/Python-CRUD-Employee-System.git
```

### 2. Navigate to the project

```bash
cd Python-CRUD-Employee-System
```

### 3. Install dependencies

```bash
pip install flask
```

### 4. Run the application

```bash
python app.py
```

### 5. Open your browser

```
http://127.0.0.1:5000
```
